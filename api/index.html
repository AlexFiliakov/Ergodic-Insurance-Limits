API Reference

This section provides detailed documentation for all modules, classes, and functions in the Ergodic Insurance Limits package.

The package implements a comprehensive framework for optimizing insurance limits using ergodic (time-average) theory, demonstrating how insurance transforms from a cost center to a growth enabler when analyzed through time averages.
